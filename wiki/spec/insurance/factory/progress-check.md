# 進度檢查

## 功能說明
* 檢查工單編號之保單進度
* 進度選擇後查詢結果將顯示此進度"以外"的保單
    * 進度種類用不同顏色表示
* 帳號未設定委外廠商此功能無法使用

## 鎖定規則
| 項目     | 說明                                                                 |
| -------- | -------------------------------------------------------------------- |
| 查詢按鈕 | 需填寫工單編號, 並符合工單編號之字數即形式 <br> 該帳號須設定委外廠商 |

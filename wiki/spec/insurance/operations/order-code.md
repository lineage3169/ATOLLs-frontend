# 生產 - 工單確認

## 功能
* 當工單跑自動化流程遇到檢查階段時將會顯示於此區
* 待使用者確認後可允許或註銷

## 欄位名稱
| 項目     | 說明                                                                           |
| -------- | ----------------------------------------------------------------------------- |
| 確認碼   | 允許或註銷時需輸入                                                                |
| 工單編號 | 遠雄: 自動化抓取資料時, 依檔名分類相同模組歸為同一工單 <br> 台壽: 依xml 檔分為一工單       |
| 里程碑   | 表示工單目前自動化流程的階段                                                        |

## 鎖定規則
| 項目       | 說明                 |
| ---------- | -------------------- |
| 註銷按鈕 | 該工單若距今不超過7天時鎖定 |

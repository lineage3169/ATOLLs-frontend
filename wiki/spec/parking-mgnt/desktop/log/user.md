# 使用者紀錄

## 功能
* 提供使用者查詢各帳號之行為紀錄
* 提供使用者列表
    * 選擇可查看該使用者之紀錄
    * 提供快速查詢過濾使用者

| 項目   | API Parameters | 說明                                                        |
| ------ | -------------- | ----------------------------------------------------------- |
| 日期   | create_time    | 該操作之日期與時間                                          |
| 層級   | level          | 0: 'NOTSET' 10: 'DEBUG' 20: 通知 30: 警告 40: 錯誤 50: 嚴重 |
| IP     | ip             | 該使用者IP                                                  |
| 使用者 | user           | 使用者名稱                                                  |
| 訊息   | message        | 該使用者之操作訊息                                          |

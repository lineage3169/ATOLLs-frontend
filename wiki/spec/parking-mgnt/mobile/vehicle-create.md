# 建立車籍

## 功能
* 車子剛到庫時提供使用者為車輛建檔之功能

## 輸入項目

| 項目     | API Parameters   | 說明                                  |
| -------- | ---------------- | ------------------------------------- |
| 生產編號 | origin_id        | 選填, 若未填將帶入底盤編號            |
| 底盤編號 | name             | 必填                                  |
| QRcode   | barcode          | 必填, 出庫後會回收並重複使用          |
| 點收     | receive          | 勾選與輸入點收項目                    |
| 待複檢   | reconfirm.status | 水險, 剛到庫就發現車輛有問題          |
| 備註     | reconfirm.remark | 發現車輛問題之備註                    |
| 附件上傳 | file             | 用於拍照配件與有問題的地方 (提供多選) |

# 車輛作業

## 功能
* 輸入生產編號末五碼
* 列出查詢結果
* 點選列表中編號
* 針對該車輛勾選或取消工作項目

## 工作項目
| 項目       | 說明         |
| ---------- | ----------- |
| PDI/洗車   | 全車檢測       |
| 隔熱紙     | 貼隔熱紙       |
| 配件      |  安裝配件      |
| 美容鍍膜   |  美容鍍膜車輛      |
